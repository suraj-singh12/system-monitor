# System Monitor For Windows 

## How To Run
+ Clone the Repo
+ Install Xampp Server
+ Run the Xampp Server (Apache and MySQL)
+ Place the cloned repo in the `htdocs` folder of the Xampp Server
+ Open the `index.php` file in the browser
